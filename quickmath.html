QuickMath

Fast calculations in the browser

Live Demo Source Code

About

QuickMath is an online math tool built with Blazor WebAssembly. It runs entirely in the browser — no server needed after the initial load. Designed for speed, simplicity, and education.

Tech Stack

Blazor WebAssembly

Runs C# code directly in the browser via WebAssembly.

C#

Core application logic and computation engine.

HTML / CSS

Responsive UI with a clean, accessible design.

GitHub Pages

Hosted and deployed via GitHub Pages with Actions.

Libraries

  • Microsoft.AspNetCore.Components.WebAssembly — Blazor WASM framework
  • Microsoft.AspNetCore.Components.WebAssembly.DevServer — development server
  • System.Net.Http.Json — HTTP JSON handling
  • Microsoft.NET.Sdk.WebAssembly — WASM SDK
